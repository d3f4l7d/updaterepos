## Usage
Just execute like the below.

1. ```chmod +x updaterepos.sh```

2. ```./updaterepos.sh 2>&1 | tee updaterepos_stdouterr.txt```
## Reinstallation
Just execute like the below.

1. ```cd ~/.config/d3f4l7d/updaterepos```

2. ```git fetch --all```

3. ```git reset --hard origin/main```
